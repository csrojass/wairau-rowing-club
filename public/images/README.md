# Images

Drop photo files here. Suggested filenames so I can wire them into the right slots:

## Priority (used right now)

- `hero.jpg` — homepage hero background (ideally wide, 1920×1080+, action shot of rowers)
- `about.jpg` — homepage "Our home" section + about page (boathouse or river view)
- `logo.svg` or `logo.png` — official Wairau Rowing Club logo (transparent background preferred)

## Programs (used on home + programs page)

- `program-club.jpg` — club rowing crew
- `program-college.jpg` — Marlborough Girls'/Boys' College rowers
- `program-corporate.jpg` — corporate 8 / team-building shot
- `program-performance.jpg` — high-performance squad

## News (will be replaced when CMS is connected)

- `news-1.jpg`, `news-2.jpg`, `news-3.jpg` — for the homepage news cards

## Optional

- `boathouse.jpg` — exterior of the clubhouse
- `river.jpg` — wide shot of the Wairau River
- `og-image.jpg` — 1200×630 social-media preview image

## File tips

- Keep photos under ~500 KB each (use https://squoosh.app to compress)
- Prefer landscape orientation
- JPG for photos, PNG/SVG for logos
